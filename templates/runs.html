1. Pull records from the portal

A scrape opens a hidden browser and reads your portal. It paces itself with long, human-like pauses to avoid tripping the city's automated-scraper alarms, so a full scrape can take hours — that's intentional. Only one scrape runs at a time.

{% if scrape_running %}scrape running…{% endif %}

2. AI analysis

Extract text from attachments, classify messages, and generate per-request summaries. Classification and summaries call the Anthropic API (your key) and may cost money; results are saved so re-runs only process what's new (cheap). Uses inexpensive Haiku for classification and reuses a cached legal reference to keep costs low.

Data backups

A snapshot of your database is saved automatically before every update, and you can save one anytime. Backups live in the backups folder. To roll back, close the program and run Restore.bat (Windows) / Restore.command (Mac) — it lists your backups and restores the one you pick (saving your current data first, so it's reversible). Downloaded attachment files aren't copied here — they're large and are never changed by an update.
